

# 一、启动与配置类核心注解

1. **@SpringBootApplication**：Spring Boot 项目启动类唯一核心注解，标注在主启动类上（内含组件扫描、自动配置功能）。
2. **@Configuration**：标注配置类（如 RedisConfig、MQConfig），替代传统 XML 配置，让 Spring 管理配置 Bean。

# 二、三个核心请求参数注解（仅 Controller 层使用）

1. **@PathVariable**：从 URL 路径中拿参数（如 `/goods/123` 拿 id=123）；
2. **@RequestParam**：从 URL 后缀（? 后）或表单拿参数（如 `/goods?name=手机`）；
3. **@RequestBody**：接收前端传的 JSON 格式数据（如创建商品时传的名称、价格）。
