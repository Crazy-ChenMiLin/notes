### （一）通用成功 / 基础错误

| 报错代码（十六进制） | 官方宏定义                  | 官方释义             | 小白大白话解读                    |
| ---------- | ---------------------- | ---------------- | -------------------------- |
| 0x00000000 | MV_OK                  | 操作成功             | 没问题，步骤执行正常                 |
| 0x80000001 | MV_E_FAIL              | 操作失败（通用错误，无具体分类） | 出问题了，但没明确说哪里出问题，需结合场景排查    |
| 0x80000002 | MV_E_INVALID_PARAMETER | 输入参数无效           | 你传的参数不对（比如相机 ID 填错、数值超出范围） |
| 0x80000003 | MV_E_OUT_OF_MEMORY     | 内存不足             | 电脑内存不够用了，无法分配资源给相机连接       |
1. 日志中出现关键错误码：



2. [0x80000221] (MV_E_STATE / MV_E_BUSY) 
	设备繁忙 / 被占用 (Device is busy)
详细说明：这个错误码最直接的含义是相机设备当前处于不可操作的状态。
后果：在高并发下，SDK 在“刚销毁”和“刚初始化”之间产生竞态条件，导致海康 SDK 报 0 x 80000221 (MV_E_STATE) 错误，无法再次打开设备。


3. [0x80000203] (MV_E_ACCESS / MV_E_NOENOUGH_PRI)
	含义：设备无访问权限
	说明：当前用户或应用程序没有足够的权限访问相机设备。可能是由于权限设置、用户账户控制
	或设备已被其他进程占用（mvs 软件打开/进程不干净）


4. [0 x 80000206]Connect to camera failed after retries, errcode
	含义：网络相关错误
	说明：更广泛的网络错误类别，包括但不限于网络延迟、丢包、路由问题等
