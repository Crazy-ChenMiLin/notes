Description:

Web server failed to start. Port 8080 was already in use.

#### 1. 检查 8080 端口占用情况

打开 CMD 或 PowerShell，执行：

cmd

```cmd
netstat -ano | findstr "8080"
```


    ```plaintext
    TCP    0.0.0.0:8080           0.0.0.0:0              LISTENING       1234  # 1234 是 PID（进程ID）
    TCP    [::]:8080              [::]:0                 LISTENING       1234
    ```
    
- 关键：记住最后一列的 **PID 数字**（如上例的 1234）。

#### 2. 结束占用 8080 端口的进程

用上面查到的 PID，执行：

```cmd
taskkill /F /PID 1234  # 把 1234 换成实际查到的 PID
```
