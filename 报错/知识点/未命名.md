自动序列化和反序列化器

``` js
你写的代码：redisTemplate.opsForValue().set("user:1001", new User(1001, "小明"));
↓
1. Key 序列化："user:1001"（String）→ 自动转 UTF-8 字节数组 → 存入 Redis 的 key；
2. Value 序列化：User 对象 → Jackson 自动转 JSON 字节数组（带类型信息）→ 存入 Redis 的 value；
↓
Redis 中存储：key="user:1001"，value="{"@class":"org.example.springredis.User","id":1001,"name":"小明"}"（字节数组形式）
↓
你写的代码：User user = (User) redisTemplate.opsForValue().get("user:1001");
↓
3. Key 反序列化：Redis 的 key 字节数组 → 自动转 String → 匹配你传入的 key；
4. Value 反序列化：Redis 的 value 字节数组 → Jackson 自动读 @class 类型 → 转成 User 对象 → 返回给你；
```

只能存入字节码
