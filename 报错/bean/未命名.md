
```java
// ❶ 关键：漏加 @Service/@Component 等注解 → 该类不是Spring Bean
public class UserService {
    // ❷ 试图注入Spring容器中的Bean
    @Autowired
    private OrderService orderService;

    public void test() {
        orderService.doSomething();
    }
}

// 正常的Spring Bean（作为被注入的目标）
@Service
public class OrderService {
    public void doSomething() {}
}

// 测试类触发报错
@SpringBootTest
public class InjectErrorTest {
    @Test
    public void testInject() {
        // ❸ 手动new对象（非Spring创建）→ 注入失效
        UserService userService = new UserService();
        userService.test();
    }
}
```

```java
@Service // 该类是Spring Bean
public class AiCodeHelper {
    // ❶ 错误：静态字段不能用@Resource/@Autowired注入
    @Resource
    private static ChatModel chatModel; 

    public void testChat() {
        chatModel.generate("test");
    }
}
```

| 问题类型            | 核心异常类型                        | 报错关键词 / 描述                                                                     | 根本原因                                |
| --------------- | ----------------------------- | ------------------------------------------------------------------------------ | ----------------------------------- |
| 注解使用错误（静态字段）    | IllegalArgumentException      | @Autowired is not supported on static fields                                   | 注解用法违反 Spring 规则                    |
| 非 Spring 类加注解   | NullPointerException          | 注入的对象为 null（无特定 Spring 报错）                                                     | 非 Spring 对象无法触发注入                   |
| 配置缩进导致 Bean 未创建 | NoSuchBeanDefinitionException | `No qualifying bean of type 'com.example` / `org.example' avai`                | **第三方 Starter 没创建这个 Bean**（配置条件不满足） |
| 扫描范围错误          | NoSuchBeanDefinitionException | `No qualifying bean of type 'dev. Langchain 4 j. Model. Chat. ChatModel' avai` | **你没让 Spring 创建这个 Bean**（漏注解 / 漏扫描  |
