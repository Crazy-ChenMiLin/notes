### 2. 编辑系统变量 PATH（关键！）

系统变量优先级更高，需手动添加 Python 3.10 路径并置顶：

1. 右键「此电脑」→「属性」→「高级系统设置」→「环境变量」；
2. 在「系统变量」区域找到「Path」，点击「编辑」；
3. 点击「新建」，依次添加以下两个路径：
    ```
    C:\Users\26487\AppData\Local\Programs\Python\Python310\Scripts\
    C:\Users\26487\AppData\Local\Programs\Python\Python310\
    ```
    
4. 选中新增的 Python 3.10 路径，点击「上移」至列表顶部；
5. 找到系统变量中 Python 3.14 的路径（如`C:\Python314\`），点击「下移」至 3.10 路径之后；
6. 点击「确定」保存系统变量，再点击「环境变量」主窗口的「确定」。

### 3. 验证用户变量（兜底）

检查「用户变量」的 PATH，确保 Python 3.10 路径同样置顶（无则补充），避免局部环境冲突。

### 4. 重启命令行 & 验证版本

环境变量修改仅对新窗口生效，需：

1. 关闭所有已打开的 PowerShell/CMD 窗口；
2. 重新打开「管理员 PowerShell」，执行验证命令：
    ```
    # 验证全局Python版本（应显示3.10.11）
    python --version
    # 验证pip关联版本
    pip --version
    ```
