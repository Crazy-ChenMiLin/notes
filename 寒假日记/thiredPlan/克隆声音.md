# GPT-SoVITS 从零到微调完整操作文档

#### **一、安装Anaconda**

1. 打开浏览器，访问Anaconda官网，下载Windows版。
    
2. 双击安装包，按提示下一步，勾选“Add Anaconda3 to my PATH environment variable”，完成安装。
    
3. 安装后，打开“Anaconda Prompt”。
    

#### **二、克隆项目仓库**

1. 在Anaconda Prompt中输入命令：`git clone https://github.com/RVC-Boss/GPT-SoVITS.git`，按回车，等待下载完成。
    
2. 进入项目文件夹：`cd GPT-SoVITS`。
    

#### **三、创建虚拟环境并安装依赖**

1. 创建环境：`conda create -n gpt-sovits python=3.10`，输入“y”确认。
    
2. 激活环境：`conda activate gpt-sovits`。
    
3. 安装依赖：`pip install -r requirements.txt`，耐心等待安装完成。
    

#### **四、准备语音样本**

1. 用Mac mini录制1分钟左右音频，文稿：“大家好，今天天气不错，我来测试一下声音克隆效果。这段文字大概60秒，语速自然，保持环境安静，尽量不要有杂音。”
    
2. 将音频保存为`wav`格式，采样率44100Hz，单声道。
    
3. 把音频文件放到项目文件夹的`dataset/your_name/`目录下。
    

#### **五、运行WebUI并微调模型**

1. 在Anaconda Prompt中输入：`python webui.py`，启动WebUI界面。
    
2. 浏览器会自动打开页面，点击左侧“训练”标签。
    
3. 在“说话人名称”处输入`your_name`，点击“预处理数据”，等待完成。
    
4. 预处理后，点击“开始训练”，根据提示选择训练参数，开始微调。
    

#### **六、用微调模型朗读文本**

1. 训练完成后，点击左侧“推理”标签。
    
2. “说话人”选择`your_name`，输入要朗读的文本，点击“生成语音”即可。
