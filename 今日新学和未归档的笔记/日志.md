# What 日志：

| 日志级别  | 优先级 | 核心含义                     | 适用场景（小白能懂的例子）                   |
| ----- | --- | ------------------------ | ------------------------------- |
| DEBUG | 低   | 调试信息，记录程序运行的关键中间状态       | 开发 / 测试时查问题（比如 “查询用户 ID=1 的参数”） |
| INFO  | 中   | 常规运行信息，记录程序正常工作的关键节点     | 生产环境必开（比如 “服务启动成功”“用户下单完成”）     |
| ERROR | 高   | 错误信息，程序某功能执行失败，但整体程序还能运行 | 比如 “数据库查询失败”“接口调用超时”            |
