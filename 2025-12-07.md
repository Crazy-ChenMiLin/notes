| 数据类型       | 单个写入操作（Spring API + Redis 命令）                                  | 单个写出操作（Spring API + Redis 命令）                                        | 过期时间设置方式（Spring API）                                                                   | 核心原因说明                                                     |
| ---------- | -------------------------------------------------------------- | -------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| **String** | `opsForValue().set(key, value)` → `SET key value`              | `opsForValue().get(key)` → `GET key`                                 | 方式 1：写入时直接设置：`set(key, value, ttl, unit)`<br><br>方式 2：写入后单独设置：`expire(key, ttl, unit)` | `SET`命令原生支持直接加过期时间（`EX`/`PX`参数），Spring API 简化了常用场景。        |
| **Hash**   | `opsForHash().put(key, field, value)` → `HSET key field value` | `opsForHash().get(key, field)` → `HGET key field`                    | 写入后单独设置：`expire(key, ttl, unit)`                                                       | `HSET`仅操作 Hash 内部字段，不支持直接设置 Key 过期时间，需通过`EXPIRE`单独设置。      |
| **List**   | `opsForList().rightPush(key, value)` → `RPUSH key value`（尾部插入） | `opsForList().rightPop(key)` → `RPOP key`（尾部弹出）                      | 写入后单独设置：`expire(key, ttl, unit)`                                                       | `RPUSH`仅操作 List 元素（尾部插入），不支持直接设置 Key 过期时间，需通过`EXPIRE`单独设置。 |
| **Set**    | `opsForSet().add(key, value)` → `SADD key value`               | `opsForSet().isMember(key, value)` → `SISMEMBER key value`（检查元素是否存在） | 写入后单独设置：`expire(key, ttl, unit)`                                                       | `SADD`仅向 Set 添加唯一元素，不支持直接设置 Key 过期时间，需通过`EXPIRE`单独设置。      |
| **ZSet**   | `opsForZSet().add(key, value, score)` → `ZADD key score value` | `opsForZSet().score(key, value)` → `ZSCORE key value`（读取元素分数）        | 写入后单独设置：`expire(key, ttl, unit)`                                                       | `ZADD`仅向 ZSet 添加带分数的元素，不支持直接设置 Key 过期时间，需通过`EXPIRE`单独设置。   |
