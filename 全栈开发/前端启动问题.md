# 前端启动问题复盘（POSTMORTEM）

**概述**

- 项目：`wxdemo-web`（Vite + Vue3 最小示例）
- 目标：把现有 `src` 代码在本地用 Vite 启动并能访问页面
- 结果：已修复多个问题，前端现在能够在本地启动并显示登录页面

---

**一、遇到的问题（按出现顺序）**

1. ***插件安装后又出现 ESM/CJS 混用错误：***

- **ESM**：是 JavaScript 现代的**模块规范**（用 `import/export` 导入导出），很多新工具（如 `@vitejs/plugin-vue`）只支持这种规范。
- **CJS**：是 Node.js 旧的模块规范（用 `require/module.exports` 导入导出）。
- 错误原因：项目里同时用了两种规范，导致工具 “不认识” 模块格式。


 2.***Vite. Config. Js***
- 模块别名解析失败：
`Failed to resolve import "@/api/login"`，导致组件里引用的 `@/...` 无法解析。


3. 依赖缺失：
如 ==`axios` 未安装==，导致 `import axios from 'axios'` 无法解析。


4.- 启动目录错误：
曾在 == `src` 子目录==启动 `npm run dev`，导致 Vite 无法正确读取根目录配置和 node_modules。


5.- 启动时出现 Vite 对 `.vue` 文件解析错误，提示未安装或未加载 `@vitejs/plugin-vue`。
