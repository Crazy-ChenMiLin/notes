## 一、前期准备（必做！避免报错）

### 1. 关闭天翼云盘「设备锁」（关键！）

如果不关闭，挂载时会提示「Device ID 不存在」，操作如下：

1. 打开手机**天翼云盘 APP** → 点击底部「我的」
2. 进入「设置」→「账号与安全」→「安全中心」
3. 找到「设备锁」，关闭开关（验证手机号后生效）

### 2. 确认 OpenList 环境

- 确保 OpenList 已正常启动（状态为「已启动」）
- 无需安装任何 PC/TV 端天翼云盘，仅需**手机 APP + OpenList 网页**

---

## 二、核心挂载步骤（3 步搞定）

==前提：OpenList 的端口号在可行范围内==


### 步骤 1：进入 OpenList 存储管理

1. 登录 OpenList 网页后台
2. 点击左侧菜单 **【存储】**
3. 点击页面顶部 **【添加】** 按钮

### 步骤 2：选择驱动并填写配置

在「驱动」下拉框中，选择：

✅ **天翼云盘客户端（189CloudPC）**

只填写以下核心项，**其他全部留空 / 保持默认**：

| 配置项       | 填写内容 & 说明                                    |
| --------- | -------------------------------------------- |
| 挂载路径      | 自定义，如 `/tianyi`（唯一不重复即可，用于在 OpenList 首页访问）   |
| 用户名 *     | 填写你**天翼云盘绑定的手机号**（和手机 APP 登录账号一致）            |
| 密码 *      | 填写你**天翼云盘的登录密码**（和手机 APP 登录密码一致）             |
| 验证码       | 点击「获取验证码」，手机收到短信后，将 6 位数字填入此处                |
| 根文件夹 ID   | 填写 `-11`（天翼云盘**个人云根目录**，挂载全部文件；自定义文件夹需另取 ID） |
| WebDAV 策略 | 选择 **「302 重定向」**（最稳定，直连云盘下载，不占用服务器资源）        |
| 类型        | 选择「个人云」（挂载自己的云盘；如需家庭云，需额外获取家庭 ID）            |

> 💡 提示：其他选项（如禁用索引、排序依据等）**全部保持默认，不要修改**，避免报错。

### 步骤 3：完成挂载

1. 确认所有必填项填写无误后，拉到页面底部，点击 **【添加】**
2. 回到 OpenList「存储管理」页面，确认 `/tianyi` 状态为 **「工作中」**
3. 点击左侧菜单 **【首页】**，找到 `/tianyi` 文件夹，点击进入
4. 挂载成功！你可以看到天翼云盘里的所有文件了！

---

## 三、挂载成功后常用操作

### 1. 上传文件到天翼云盘

- 进入 `/tianyi` 文件夹
- 点击页面右侧 ** 向上箭头（↑）** 按钮，选择本地安装包上传
- 文件会自动同步到你手机天翼云盘

### 2. 下载 / 分享文件

- **下载**：点击云盘文件，直接下载到本地
- **分享**：右键文件 → 选择「生成分享链接」，可设置有效期 / 密码，复制链接发给他人即可

### 3. 挂载指定文件夹（可选）

如果不想挂载整个个人云，只想挂载某个文件夹：

1. 手机打开天翼云盘 APP → 进入目标文件夹
2. 点击右上角「分享」→「复制链接」
3. 粘贴链接到便签，提取 `folder/` 后的字符串（即为文件夹 ID）
4. 回到 OpenList 存储编辑页面，将「根文件夹 ID」替换为该字符串，保存后刷新即可
