# 1Panel 完整安装教程（Ubuntu 24.04 LTS 服务器版）

---
## 一、安装前准备

### 1. 环境要求

- 系统：Ubuntu 18.04+ / Debian 10+ / CentOS 7+（你的 Ubuntu 24.04 完全兼容）
- 架构：x86_64 /aarch64（你的服务器是 aarch64，支持）
- 权限：拥有 **root 权限**（你当前已登录为 root）
- 网络：服务器可访问公网，且防火墙 / 安全组后续需放行面板端口

### 2. 工具准备

- SSH 连接工具（如 Xshell、FinalShell、Windows Terminal 等，你已在用 Xshell）
- 浏览器（用于后续访问面板）

---
## 二、一键安装 1Panel

### 1. 执行安装命令

在 SSH 终端中，复制粘贴以下**完整命令**并回车：

```
curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && sudo bash quick_start.sh
```

### 2. 安装过程交互

脚本会自动检测系统并开始安装，中途会提示你配置以下信息（直接回车可使用默认值）：

1. **设置面板端口**：默认 `23751`，建议直接回车（若需自定义，输入数字后回车）
2. **设置面板用户名**：默认 `1panel`，可自定义后回车
3. **设置面板密码**：输入自定义密码（输入时不显示，输完回车后再次确认）

### 3. 安装成功提示

安装完成后，终端会输出类似信息（**务必截图保存**）：

```
1Panel 安装成功！
面板地址: https://你的服务器公网IP:23751
用户名: 1panel
密码: 你设置的密码
```

> 若提示「1Panel 已安装，请勿重复安装」，说明你的服务器已完成安装，直接跳至后续步骤。

---

## 三、放行防火墙端口（关键！否则无法访问面板）

### 1. Ubuntu 系统防火墙放行

你的服务器是 Ubuntu 24.04，执行以下命令放行 1Panel 端口（以默认 `23751` 为例）：

```
# 放行端口
sudo ufw allow 23751/tcp
# 重载防火墙规则
sudo ufw reload
# 验证规则是否生效
sudo ufw status
```

> 若你自定义了端口，将 `23751` 替换为你的实际端口号。

### 2. 云服务器安全组放行（若为云服务器）

如果你的服务器是阿里云 / 腾讯云 / 华为云等云厂商的实例，还需要：

1. 登录云厂商控制台 → 找到对应服务器 → 进入「安全组」
2. 添加**入方向规则**：
    
    - 协议：TCP
    - 端口：`23751`（或你的自定义端口）
    - 授权对象：`0.0.0.0/0`（允许所有 IP 访问，或根据需求限制 IP）
    
3. 保存规则，等待 1-2 分钟生效

## 四、获取登录信息（若已安装但忘记）

### 1. 查看用户名

```
1pctl user-info
```


```
Username: 1panel
```

### 2. 重置密码

若忘记密码，执行以下命令重置：


```
1pctl reset
```

按提示输入新密码，完成后立即生效。

### 3. 查看面板端口


```
cat /opt/1panel/conf/1panel.conf | grep HttpPort
```

输出示例：

```
HttpPort = 23751
```

等号后的数字即为访问端口。

### 4. 检查服务状态


```
1pctl status core
```

- 若显示 `active (running)`：服务正常运行
- 若显示 `inactive`：执行启动命令
    
    ```
    1pctl start core
    ```
    

---

## 五、登录 1Panel 面板

### 1. 访问面板地址

打开浏览器，输入地址（替换为你的实际信息）：

plaintext

```
https://你的服务器公网IP:端口号
```

示例（你的服务器）：

```
https://146.56.102.26:23751
```

> 注意：必须使用 `https` 协议，浏览器会提示「不安全」，点击「高级」→「继续访问」即可。

### 2. 登录面板

输入获取到的**用户名**和**密码**，点击「登录」，即可进入 1Panel 管理界面。
