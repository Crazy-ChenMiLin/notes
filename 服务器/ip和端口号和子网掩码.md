# 1 . 概念

- **IP 地址**：你家的门牌号（比如`192.168.100.128`），别人找你靠这个号。
- **子网掩码（比如 / 24 对应 255.255.255.0）**：小区的围墙范围 —— 告诉你 “哪些门牌号和你在同一个小区”（比如 192.168.100 开头的，都是你小区邻居，能直接串门）。
- **网关**：小区的大门（一般是路由器 IP，比如`192.168.100.1`）—— 你要去别的小区（其他网段），必须从这个大门出去。

### 1. 第一个网卡：`lo`（回环网卡）

回环网卡是系统 “本地通信” 用的虚拟网卡，仅本机内部可用：

- 状态标识：`<LOOPBACK,UP,LOWER_UP>`
    - `LOOPBACK`：回环类型（虚拟网卡）
    - `UP`：网卡已启用
    - `LOWER_UP`：网卡物理层（逻辑层）已连通
- `mtu 65536`：最大传输单元（回环网卡的 MTU 通常很大）
- 链路信息：`link/loopback 00:00:00:00:00:00` → 回环网卡的 MAC 地址固定为全 0
- IPv4 地址：`inet 127.0.0.1/8` → 本地回环地址（仅本机访问）
- IPv6 地址：`inet6 ::1/128` → IPv6 版本的本地回环地址

### 2. 第二个网卡：`ens33`（物理 / 虚拟机网卡）

这是服务器对外通信的实际网卡（比如虚拟机的网卡）：

- 状态标识：`<BROADCAST,MULTICAST,UP,LOWER_UP>`
    - `BROADCAST`：支持广播通信
    - `MULTICAST`：支持组播通信
    - `UP`/`LOWER_UP`：网卡已启用且物理层连通
- `mtu 1500`：以太网默认最大传输单元
- 链路信息：`link/ether 00:0c:29:13:d8:de` → 网卡的 MAC 地址（硬件地址），广播地址是`ff:ff:ff:ff:ff:ff`
- IPv4 地址：`inet 192.168.100.128/24` → 服务器的局域网 IP（对外通信的地址）
- IPv6 地址：`inet6 fe80::4f7f:c18a:89d9:346c/64` → 本地链路级的 IPv6 地址
