# 基本数据类型有 ：
①、数值型

- 整数类型（byte、short、int、long）
- 浮点类型（float、double）

②、字符型（char）

③、布尔型（boolean）

它们的默认值和占用大小如下所示：

| 数据类型    | 默认值      | 大小         |
| ------- | -------- | ---------- |
| boolean | false    | 1 字节或 4 字节 |
| char    | '\u0000' | 2 字节       |
| byte    | 0        | 1 字节       |
| short   | 0        | 2 字节       |
| int     | 0        | 4 字节       |
| long    | 0L       | 8 字节       |
| float   | 0.0f     | 4 字节       |
| double  | 0.0      | 8 字节       |
| String  | Null     |            |
String 不是数据类型，|**Java**|引用类型（类类型）
