# When

| 数据类型    | 一句话用途（像说人话一样）                                  | 举个实际例子                                   |
| ------- | ---------------------------------------------- | ---------------------------------------- |
| byte    | ==视频、图片、压缩包==                       这些 “非文字文件” | 用字节数组读手机里的.jpg 图片、.mp4 视频                |
| short   | 内存不够时存中等大小的整数（比 int 省空间）                       | 传感器采集的温度（-50~100℃）、大量短数字存储               |
| int     | 最常用的==整数类型==                                   | 统计商品销量、用户 ID、循环次数（for (int i=0;...)）     |
| long    | 存超大的数，比如时间、大 ID                                | 记录时间戳（比如文件创建时间）、雪花算法生成的唯一 ID             |
| float   | 存小数但不用太精准（省内存）                                 | 游戏里角色的坐标（x:123.4f, y:56.7f）、天气湿度（60.2f%） |
| double  | 日常用的小数（比 float 精准，默认选它）                        | 商品价格（99.99）、身高（1.75 米）、科学计算的小数结果         |
| char    | 存单个字符（中文、英文、符号都行）                              | 统计 “a” 在字符串中出现的次数、输入框里的单个字符              |
| boolean | 记 “是 / 否”“开 / 关”                               | 用户是否登录（true = 登录）、订单是否支付（false = 未支付）    |
| String  | 存文字、句子（人类能直接看懂的）                               | 用户名（“张三”）、商品描述（“高清摄像头”）、JSON 文本          |


String 不是数据类型，|**Java**|引用类型（类类型）
