# 1. # Long 型转 string：
时间：Long
大数：Long
	 拼接空字符串（""）（可行但不推荐）
````java
long num = 100000L;
System.out.println(num+""); // 输出：100000
````

	String.valueOf (long)（最推荐）
`
```java
long num = 123456789L;
// 正确转换：纯数字字符串
String str = String.valueOf(num); 
System.out.println(str); // 输出：123456789
```
`
	 Long.toString (long)（专门针对 long 类型）
```java
long num = 987654321L;
String str = Long.toString(num);
System.out.println(str); // 输出：987654321
```
