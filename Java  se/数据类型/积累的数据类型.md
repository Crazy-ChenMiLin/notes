### 一、先明确：特殊情况 - 无返回值（void）

### 二、有返回值：核心分为 2 大类


#### 1. 基本数据类型（8 种）

这是最基础的返回值类型，用于存储原始的简单数据，采用**值传递**方式（返回的是数据的副本）。

| 分类   | 具体类型                         | 示例场景          |
| ---- | ---------------------------- | ------------- |
| 整数类型 | byte、short、int（interge）、long | 返回计数结果、ID 等   |
| 浮点类型 | float、double                 | 返回金额、精度数值等    |
| 布尔类型 | boolean                      | 返回判断结果（是 / 否） |
| 字符类型 | char （String）                | 返回单个字符（如性别标识） |


#### 2. 引用数据类型（核心高频类型，采用引用传递）

引用数据类型返回的是**对象的内存地址引用**，而非对象本身，是开发中使用最多的返回值类型，常见细分如下：

##### （1）自定义类实例
	如：User-Result
```java
// 自定义实体类（引用类型）
public class User {
    private String name;
    private Integer age;
    // 构造器、getter/setter省略
}

// 返回自定义类User的实例（引用数据类型）
public User getLoginUser(String username) {
    User user = new User();
    user.setName(username);
    user.setAge(25);
    return user; // 返回User对象的引用
}


public class Result {  
    private Boolean success;  
    private String errorMsg;  
    private Object data;  
    private Long total;



```

##### （2）接口类型

```java
// 接口定义
public interface OrderService {
    void createOrder();
}

// 接口的手动实现类
public class OrderServiceImpl implements OrderService {
    @Override
    public void createOrder() {
        System.out.println("创建订单");
    }
}

// 返回接口类型（实际返回手动实现类实例）
public OrderService getOrderService() {
    return new OrderServiceImpl(); 
    // 也可以返回动态代理生成的实例（如LangChain4j场景）
}
```

##### （3）数组类型

```java
// 返回基本类型数组（int[]）
public int[] getScoreList() {
    return new int[]{90, 85, 95, 78};
}

// 返回引用类型数组（String[]）
public String[] getNameList() {
    return new String[]{"张三", "李四", "王五"};
}
```

##### （4）集合类型（Java 集合框架）

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

// 返回List集合（存储User对象）
public List<User> getUserList() {
    List<User> userList = new ArrayList<>();
    userList.add(new User("张三", 22));
    userList.add(new User("李四", 24));
    return userList;
}

// 返回Map集合（键值对数据）
public Map<String, Object> getOrderInfo() {
    Map<String, Object> orderMap = new HashMap<>();
    orderMap.put("orderId", "O20260104001");
    orderMap.put("price", 99.9);
    orderMap.put("status", "已支付");
    return orderMap;
}
```


##### （6）包装类型

对应 8 种基本数据类型的包装类（如 `Integer`、`Boolean`、`Double` 等），常用于集合存储、框架参数传递等场景。

```java
// 返回Integer包装类型
public Integer getUserId() {
    return 1001; // 自动装箱（int → Integer）
}

// 返回Double包装类型
public Double getProductPrice() {
    return 199.99;
}
```

##### （7）其他特殊引用类型

- 枚举类型（Enum）：返回预定义的枚举常量，用于状态标识等场景；
- `null`：表示返回空引用（无实际对象），所有引用类型都可以返回 `null`。

```java
// 枚举定义
public enum OrderStatus {
    UNPAID, PAID, SHIPPED, RECEIVED
}

// 返回枚举类型
public OrderStatus getOrderStatus(String orderId) {
    return OrderStatus.PAID;
}

// 返回null（引用类型特有）
public User getNullUser() {
    return null; // 无实际User对象返回
}
```
