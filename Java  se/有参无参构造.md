# 一.项目中
@data 有参数代替 get set
@NoArgsConstructor  
@AllArgsConstructor

# Why
- 知道值 → 有参构造（直接传值，一步到位）；
- 不知道值 → 无参构造（先建空对象，后续赋值）

``` java
import java.util.LinkedList;
import java.util.Queue;

class MyStack {
    Queue<Integer> queue1; // 核心队列：队首=栈顶
    Queue<Integer> queue2; // 临时队列：push时中转

    // 需求1：有参构造 → 创建栈时直接传入初始元素
    public MyStack(int x) {
        // 第一步：先初始化两个空队列（基础操作，和无参构造一样）
        queue1 = new LinkedList<>();
        queue2 = new LinkedList<>();
        // 第二步：把初始元素x放入栈（复用push逻辑，保证栈的规则）
        this.push(x); // 调用自己的push方法，让x成为栈顶
    }

    // 需求2：无参构造 → 创建空栈（保留原逻辑）
    public MyStack() {
        queue1 = new LinkedList<>();
        queue2 = new LinkedList<>();
    }

    // 原push逻辑（不变）
    public void push(int x) {
        queue2.offer(x);
        while(!queue1.isEmpty()){
            queue2.offer(queue1.poll());
        }
        Queue<Integer> temp=queue1;
        queue1=queue2;
        queue2=temp;
    }

    // 原pop逻辑（不变）
    public int pop() {
        return queue1.poll();
    }

    // 原top逻辑（不变）
    public int top() {
        return queue1.peek();
    }

    // 原empty逻辑（不变）
    public boolean empty() {
        return queue1.isEmpty();
    }
}
```

#### 需求 1 的使用示例（创建时就有元素 1，后续加其他值）

 ``` java
public class Test {
    public static void main(String[] args) {
        // 1. 有参构造创建栈，初始元素就是1（你知道初始值，所以用有参）
        MyStack stackWithInit = new MyStack(1);
        System.out.println("初始栈顶：" + stackWithInit.top()); // 输出1
        
        // 2. 后续继续往栈里加元素（比如2、3）
        stackWithInit.push(2);
        stackWithInit.push(3);
        System.out.println("加完2、3后的栈顶：" + stackWithInit.top()); // 输出3
    }
}
```


#### 需求 2 的使用示例（空栈创建，后续赋值）
 ``` java
public class Test {
    public static void main(String[] args) {
        // 1. 无参构造创建空栈（不知道初始值，先建空架子）
        MyStack emptyStack = new MyStack();
        System.out.println("空栈是否为空：" + emptyStack.empty()); // 输出true
        
        // 2. 后续需要时，再往栈里加元素（比如1、2）
        emptyStack.push(1);
        emptyStack.push(2);
        System.out.println("加完1、2后的栈顶：" + emptyStack.top()); // 输出2
    }
}
```
