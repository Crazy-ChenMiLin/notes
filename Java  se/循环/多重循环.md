# 多重循环
```java
    public static void main(String[] args) {
        int score = 75; // 测试值：中等
        String level;

        // 四重分支：条件1→条件2→条件3→else
        if (score >= 90) {          // 条件1：优秀（90+）
            level = "优秀";
        } else if (score >= 80) {   // 条件2：良好（80-89，隐含<90）
            level = "良好";
        } else if (score >= 70) {   // 条件3：中等（70-79，隐含<80）
            level = "中等";
        } else {                    // 兜底：及格/不及格（<70）
            level = "及格/不及格";
        }

        System.out.println(level); // 输出：中等
    }
}
```
