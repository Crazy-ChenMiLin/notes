1. Spring 的依赖注入。
@Component 注解的作用是将 LoginFormDTO 注册为 Spring 容器中的一个 Bean，
这样可以通过 Spring 的依赖注入机制来管理这个类的实例。


2. @RequestBody 的工作原理是：
 Spring MVC 使用 HttpMessageConverter（特别是 Jackson 库）将请求体中的 JSON 数据转换为 Java 对象
 这个过程中，Spring 会直接通过反射机制创建对象实例，而不是从 Spring 容器中获取 Bean
创建的对象实例用于承载请求数据，然后传递给控制器方法

Spring MVC 框架看到 @RequestBody 注解后，会自动创建 LoginFormDTO 对象实例
将请求体中的 JSON 数据填充到该对象的字段中
将这个填充好的对象赋值给 loginForm 变量
因此您可以直接调用 loginForm.GetCode () 方法
