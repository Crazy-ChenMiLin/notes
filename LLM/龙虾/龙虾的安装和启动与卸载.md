# 卸载：
### 第一步：停止所有 OpenClaw 进程（先停服务，避免卸载失败）

1. 打开 Windows PowerShell（管理员身份）；
2. 执行以下命令，终止所有 OpenClaw 相关进程：
    
    powershell
    
    ```
    # 查找并终止 OpenClaw 进程
    taskkill /f /im node.exe | findstr "openclaw"
    # 如果之前安装过后台服务，停止并删除服务
    openclaw gateway stop 2>nul
    openclaw gateway uninstall 2>nul
    ```
    
    （如果提示 “找不到命令”，说明没有安装后台服务，直接跳过即可）

### 第二步：核心卸载（npm 全局移除 OpenClaw）

这是卸载程序本体的关键步骤，执行：

powershell

```
# 全局卸载 OpenClaw 包（核心命令）
npm uninstall -g openclaw
# 验证是否卸载成功（执行后提示“openclaw 不是内部或外部命令”即成功）
openclaw --version
```
