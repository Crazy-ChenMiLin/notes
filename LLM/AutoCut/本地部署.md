# 改
### 修改 md 文档：给 ai


规则：
1. **总开关（第一行）**：
    
    - 旧版：`- [ ] <-- Mark if you are done editing.`（空框，表示 “还没编辑完”）
    - 新版：`- [x] <-- Mark if you are done editing.`（打勾，表示 “编辑完成，可以剪辑了”）
    
2. **每一条字幕的保留标记**：
    
    - 旧版：所有字幕行都是 `- [ ]`（空框，表示 “这条片段我不要”）
    - 新版：所有字幕行都是 `- [x]`（打勾，表示 “这条片段我要保留”）
    

AutoCut 的规则是：**只有被打勾 `- [x]` 的片段才会被保留，空框 `- [ ]` 的会被删掉**。如果所有片段都是空框，它就找不到任何要保留的内容，直接报错。
