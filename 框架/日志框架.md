### 三者关系与对比
一、@Slf 4 j
① 角色：快捷工具；
② 核心作用：自动生成 Logger 实例，简化代码；
③ 典型使用方式：类上添加 @Slf 4 j，直接用 log.Info ()；
	

二、Logger
① 角色：操作接口；
② 核心作用：定义日志方法，提供操作能力；
③ 典型使用方式：调用 log.Info ("xxx")；

三、LoggerFactory
① 角色：实例工厂；
② 核心作用：创建 Logger 实例，关联实现；
③ 典型使用方式：Logger log = LoggerFactory.GetLogger (XXX. Class)；

四、代码示例:// ==DEBUG 级别，需要配置才能输出==
```java
# @Slf4j // 使用 SLF4J
public class PublisherTest {
  @Test
  public void testConfirmCallback() {
    log.info("进入 testConfirmCallback"); // INFO 级别，总会输出
    log.debug("收到 confirm callback 回执"); 
  }
}
```
