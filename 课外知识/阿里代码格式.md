使用 org. Slf 4 j. Logger 替代 System. Out/err。
不使用大量 static 可测试性更好（对工具类可保留静态纯函数）。
参数和常量通过 RecognitionProperties 注入，便于统一管理。
对外 API 返回 DTO（含识别字符串、置信度、原始检测列表），而不是裸 String，便于前端展示和调试。
给每个组件写单元测试：ImagePreprocessor（scale/pad）、TensorConverter（buffer 长度/顺序）、OutputParser（各种输出 shape）、NMSUtils（iou 场景）、OrtSessionManager（模拟 model file not found）。
